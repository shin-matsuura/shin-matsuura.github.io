# shin-matsuura.github.io

Personal academic website of **Shin Matsuura**, PhD student in Philosophy at the CUNY Graduate Center.

Hosted via [GitHub Pages](https://pages.github.com/) at **https://shin-matsuura.github.io**.

## Structure

| File | Description |
|------|-------------|
| `index.html` | Main page (hero, research, papers, teaching, contact) |
| `style.css` | All styles — light/dark theme, responsive layout |
| `profile.jpg` | Portrait photo (add your own) |
| `cv.pdf` | Curriculum vitae (add your own) |

## Local preview

Open `index.html` directly in any browser, or serve the directory with a local HTTP server:

```bash
npx serve .
# then visit http://localhost:3000
```

## Customization

- **Email** — replace `your.email@example.com` in `index.html`
- **Portrait** — add `profile.jpg` to the repository root
- **CV** — add `cv.pdf` to the repository root
- **Papers / Teaching** — edit the relevant `<section>` blocks in `index.html`
